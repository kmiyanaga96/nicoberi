## 1. はじめに

NicoBeri（ニコベリ）は、児童の来所スケジュール管理・出退勤打刻・送迎記録などを行うためのシステムです。

パソコンやスマートフォンのブラウザからご利用いただけます。

### このマニュアルの表記について

| 表記 | 意味 |
|------|------|
| **太字** | 画面上のボタン名・ラベル名を示します |
| `等幅文字` | 入力する値の例を示します |
| > **ヒント** | 操作のコツや補足情報です |
| > **注意** | 特にご注意いただきたい事項です |

## 2. ログイン・ログアウト

### ログインする

1. ブラウザでシステムのURLにアクセスしてください。
2. ログイン画面が表示されます。
3. **スタッフID** 欄に、管理者から通知されたIDを入力してください。
4. **パスワード** 欄にパスワードを入力してください。
5. **ログイン** ボタンを押してください。

```
┌─────────────────────────┐
│       NicoBeri           │
│   ニコベリ基幹システム     │
│                          │
│  スタッフID  [________]  │
│  パスワード  [________]  │
│                          │
│      [ ログイン ]         │
└─────────────────────────┘
```

> **注意**  
> ログインに失敗すると、画面上部に赤色のエラーメッセージが表示されます。  
> IDやパスワードを何度入力しても正しくログインできない場合は、管理者にご連絡ください。

### ログアウトする

1. ダッシュボード画面の右上にある **ログアウト** ボタンを押してください。
2. ログイン画面に戻ります。

> **ヒント**  
> 業務終了後は、必ずログアウトを行ってください。

## 3. ダッシュボードの見かた

ログインすると **ダッシュボード** 画面が表示されます。画面上部にタブが並んでおり、目的に応じてタブを切り替えて操作します。

### 画面構成

```
┌─────────────────────────────────────────────────┐
│ ダッシュボード                [スタッフ名] [ログアウト] │
│ 2026年4月11日(金)                                │
├─────────────────────────────────────────────────┤
│  [ 予定 ]   [ 月次履歴 ]   [ 児童・住所 ]          │
├─────────────────────────────────────────────────┤
│                                                  │
│  （選択したタブの内容がここに表示されます）            │
│                                                  │
└─────────────────────────────────────────────────┘
```

### メインタブ一覧

| タブ名 | 内容 |
|--------|------|
| **予定** | 今日・明日の来所予定の確認、打刻、予定の登録・キャンセル |
| **月次履歴** | 当月の打刻記録の閲覧 |
| **児童・住所** | 登録されている児童と施設の情報を確認 |

## 4. 予定タブ ─ 今日・明日の来所予定を確認する

### サブタブについて

**予定** タブを選択すると、さらに3つのサブタブが表示されます。

| サブタブ | 内容 |
|----------|------|
| **今日** | 今日の来所予定の一覧と打刻操作 |
| **明日** | 明日の来所予定の一覧と打刻操作 |
| **予定登録・キャンセル** | 来所予定の新規追加とキャンセル |

### 予定一覧の見かた

**今日** または **明日** サブタブを開くと、その日に来所が予定されている児童が一覧で表示されます。

各児童のカードには、以下の情報が表示されます。

```
┌───────────────────────────────────────────────────────┐
│  [姓名]  山田 太郎                                      │
│                                                        │
│  [迎え] [送り]     到着 --:--  (打刻)   退出 --:--      │
│                                                        │
│              ▶ 手動で修正・入力                           │
└───────────────────────────────────────────────────────┘
```

| 表示項目 | 説明 |
|----------|------|
| 姓名 | 児童のお名前です |
| キャンセル（赤ラベル） | キャンセル済みの予定にはこのラベルが付きます |
| 迎え / 送り | 送迎のフラグです（オン/オフを切り替えられます） |
| 到着 / 退出 | 打刻時刻です |

> **ヒント**  
> 予定が登録されていない場合は「来所予定は登録されていません。」というメッセージが表示されます。

## 5. 到着・退出の打刻を行う

児童が来所したとき・退出するときに、打刻を行います。

### 到着の打刻

1. **予定** タブ > **今日** サブタブを開きます。
2. 該当する児童のカードを探します。
3. **到着** 欄の右側にある **時計アイコンのボタン** を押してください。
4. 現在時刻が自動で記録され、到着欄に時刻が表示されます。
5. 緑色のチェックマークが表示されたら、打刻完了です。

### 退出の打刻

1. 到着の打刻が済んでいる児童のカードで、**退出** 欄の右側にある **時計アイコンのボタン** を押してください。
2. 現在時刻が自動で記録され、退出欄に時刻が表示されます。
3. 緑色のチェックマークが表示されたら、打刻完了です。

```
   操作の流れ:

   到着打刻前:    到着 --:--  [時計ボタン]    退出 --:--
        ↓ 到着ボタンを押す
   到着打刻後:    到着 10:00  ✅            退出 --:--  [時計ボタン]
        ↓ 退出ボタンを押す
   退出打刻後:    到着 10:00  ✅            退出 15:30  ✅
```

> **注意**  
> - 到着の打刻を行わないと、退出の打刻ボタンは表示されません。  
> - 打刻はボタンを押した瞬間の時刻が記録されますので、お子さまが来所・退出したタイミングで速やかに操作を行ってください。

## 6. 送迎フラグを切り替える

児童の送り迎えの有無を記録できます。

1. 該当する児童のカードで、**迎え** ボタンまたは **送り** ボタンを押してください。
2. ボタンの色が変わり、フラグのオン/オフが切り替わります。

| ボタンの状態 | 意味 |
|-------------|------|
| 色付き（水色 / 紫色） | フラグがオンです（送迎ありと記録されています） |
| グレー | フラグがオフです（送迎なしです） |

> **ヒント**  
> もう一度同じボタンを押すと、フラグをオフに戻すことができます。

## 7. 打刻時刻を手動で修正する

打刻ボタンを押すタイミングを逃してしまった場合など、時刻を手動で修正できます。

> **注意**  
> 一般スタッフの方は、**当日分の打刻のみ** 修正が可能です。過去の日付の打刻を修正する必要がある場合は、管理者にご依頼ください。

1. 該当する児童のカードの下部にある **「手動で修正・入力」** をクリックして開きます。
2. 修正パネルが展開されます。

```
┌──────────────────────────────────────┐
│  手動で修正・入力                       │
├──────────────────────────────────────┤
│  到着  [ 10:00 ]  [ 変更を保存 ]       │
│  退出  [ 15:30 ]  [ 変更を保存 ]       │
└──────────────────────────────────────┘
```

3. **到着** または **退出** の時刻入力欄に、正しい時刻を入力してください。
4. 入力が終わったら、対応する **変更を保存** ボタンを押してください。

> **ヒント**  
> 到着と退出はそれぞれ個別に保存されます。片方だけの修正でも問題ありません。

## 8. スタッフ共有事項（メモ）を編集する

児童に対するスタッフ間の共有事項（メモ）がある場合、児童カード上にオレンジ色のラベルで表示されます。

### メモを編集する

1. 児童カード上のオレンジ色のメモラベルをクリックすると、編集パネルが開きます。
2. テキスト欄の内容を編集してください。
3. **保存** ボタンを押して変更を確定してください。

> **ヒント**  
> このメモはスタッフ全員が閲覧できます。個人的な内容ではなく、業務に関連する共有事項を記入してください。

## 9. 予定登録・キャンセル

### 新しい来所予定を登録する

1. **予定** タブ > **予定登録・キャンセル** サブタブを開きます。
2. 「新規来所予定の追加」フォームが表示されます。

```
┌──────────────────────────────────────────┐
│  新規来所予定の追加                         │
├──────────────────────────────────────────┤
│  児童  [ プルダウンから選択 ▼ ]              │
│  日付  [ プルダウンから選択 ▼ ]              │
│                    [ 予定を追加 ]           │
└──────────────────────────────────────────┘
```

3. **児童** のプルダウンから、該当する児童を選択してください。
4. **日付** のプルダウンから、来所予定日を選択してください（本日から30日先まで選択可能です）。
5. **予定を追加** ボタンを押してください。

> **注意**  
> 同じ児童の同じ日付に、二重に予定を登録することはできません。

### 登録済みの予定を確認する

フォームの下に「確定済みの予定一覧」がアコーディオン形式で表示されます。

- 日付をクリックすると、その日の予定一覧が展開されます。
- 日付の右側に表示される数字は、その日の予定人数です。

### 予定をキャンセルする

1. 「確定済みの予定一覧」から、該当する日付をクリックして開きます。
2. キャンセルしたい児童の行にある **キャンセル** ボタン（赤い X マーク）を押してください。
3. 予定が取り消されます。

## 10. 月次履歴を確認する

当月の打刻履歴を閲覧できます。

1. **月次履歴** タブを開きます。
2. 日付ごとにアコーディオン形式で履歴がまとめられています。
3. 確認したい日付をクリックすると、その日の打刻記録が展開されます。

### 履歴の見かた

各児童について、以下の情報が表示されます。

| 項目 | 説明 |
|------|------|
| 児童名 | 児童のお名前 |
| 到着 | 到着打刻の時刻 |
| 退出 | 退出打刻の時刻 |
| 迎え / 送り | 送迎フラグ（記録がある場合のみ表示） |
| キャンセル | キャンセル済みの場合は赤いラベルが表示されます |

> **ヒント**  
> - 日曜日は赤色、土曜日は青色で日付が表示されますので、曜日の確認に便利です。  
> - 一般スタッフの方は閲覧のみとなります。過去の履歴の修正が必要な場合は管理者にご依頼ください。

## 11. 児童・住所タブ ─ 児童情報を閲覧する

登録されている児童と施設の一覧を確認できます。

1. **児童・住所** タブを開きます。
2. 「預かり所・学校の登録」セクションと「登録済み児童名簿」セクションが表示されます。

### 施設情報を確認する

- 「預かり所・学校の登録」をクリックして開くと、登録されている施設の一覧が表示されます。
- 各施設の名前と住所を確認できます。

### 児童情報を確認する

- 「登録済み児童名簿」に児童の一覧が表示されます。
- 児童の名前をクリックすると、詳細情報が展開されます。

| 項目 | 説明 |
|------|------|
| 姓 / 名 | お名前 |
| セイ / メイ | フリガナ |
| 生年月日 | 生年月日 |
| 性別 | 性別 |
| 受給者証番号 | 受給者証番号 |
| 支援区分 | 支援区分レベル |
| スタッフ共有事項 | スタッフ間の共有メモ |

> **注意**  
> 児童情報の登録・編集・削除は管理者のみ行えます。情報に誤りがある場合は管理者にご連絡ください。

## 12. よくあるご質問（FAQ）

### Q. ログインできません

以下の点をご確認ください。

- スタッフIDとパスワードが正しいか（大文字・小文字の違いにもご注意ください）
- 半角英数字で入力しているか
- アカウントが凍結されていないか（「アカウントが無効です」と表示される場合は管理者に連絡してください）

### Q. 打刻ボタンを押し忘れました

当日の分であれば、「手動で修正・入力」から時刻を直接入力して保存できます。操作方法は「[7. 打刻時刻を手動で修正する](#7-打刻時刻を手動で修正する)」をご参照ください。過去の日付の場合は管理者にご依頼ください。

### Q. 間違えて到着打刻をしてしまいました

手動修正で時刻を書き換えることが可能です。予定自体を取り消したい場合は、管理者に削除を依頼してください。

### Q. 同じ児童の予定を二重に登録しようとしたらエラーが出ました

同一日に同一児童の予定を重複して登録することはできない仕様です。既に登録されていないか「確定済みの予定一覧」をご確認ください。

### Q. 送迎フラグを間違えて押してしまいました

もう一度同じボタンを押すと、フラグをオフに戻せます。

## 13. 困ったときは

上記の内容で解決しない問題が発生した場合は、管理者にお問い合わせください。

お問い合わせ時には、以下の情報をお伝えいただくとスムーズです。

- どの画面で問題が起きたか
- どのような操作をしたか
- 画面に表示されているエラーメッセージ（もしあれば）
- 操作した日時
