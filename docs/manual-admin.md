## 1. はじめに ─ 管理者と一般スタッフの違い

管理者は、一般スタッフが利用できるすべての機能に加えて、以下の追加機能をご利用いただけます。

| 機能 | 一般スタッフ | 管理者 |
|------|:---:|:---:|
| 当日の打刻・送迎記録 | ○ | ○ |
| 来所予定の登録・キャンセル | ○ | ○ |
| スタッフ共有事項の編集 | ○ | ○ |
| 月次履歴の閲覧 | ○（閲覧のみ） | ○（編集も可） |
| 打刻の手動修正（当日分） | ○ | ○ |
| 打刻の手動修正（過去分） | × | ○ |
| 予定の削除 | × | ○ |
| 児童情報の登録・編集・削除 | × | ○ |
| 施設情報の登録・編集・削除 | × | ○ |
| スタッフアカウントの管理 | × | ○ |
| 月次実績記録表の Excel 出力 | × | ○ |
| 管理者パネルへのアクセス | × | ○ |

## 2. 管理者パネルへのアクセス方法

1. ダッシュボードにログインします（ログイン手順はスタッフ用マニュアルをご参照ください）。
2. 画面右上のヘッダー部分に **管理者パネル** というリンクが表示されています。
3. **管理者パネル** をクリックしてください。

```
┌──────────────────────────────────────────────────┐
│ ダッシュボード     [管理者パネル] [スタッフ名] [ログアウト] │
└──────────────────────────────────────────────────┘
                       ↑ ここをクリック
```

4. 管理者パネルが開きます。暗い背景の専用画面に切り替わります。

### ダッシュボードに戻る

管理者パネルの右上にある **ダッシュボードに戻る** ボタンを押すと、通常のダッシュボードに戻ります。

### 管理者パネルのタブ構成

| タブ名 | 内容 |
|--------|------|
| **スタッフ管理** | スタッフアカウントの一覧・作成・編集・凍結・削除 |
| **財務** | 会計関連機能（現在は準備中です） |

> **ヒント**  
> 管理者パネルの **スタッフ管理** タブの下部に、Excel出力機能も配置されています。

## 3. スタッフ管理

### 3-1. スタッフ一覧を確認する

1. 管理者パネルを開きます。
2. **スタッフ管理** タブを選択します（初期状態で選択されています）。
3. 登録されているスタッフの一覧がカード形式で表示されます。

各カードには以下の情報が表示されます。

| 表示 | 説明 |
|------|------|
| 氏名 | スタッフの名前 |
| ロールラベル | `admin`（管理者）または `staff`（一般スタッフ） |
| 凍結済み（赤い警告） | アカウントが凍結されている場合に表示されます |

カードをクリックすると、詳細な編集フォームが展開されます。

---

### 3-2. スタッフ情報を編集する

1. 編集したいスタッフのカードをクリックして、詳細を展開します。
2. 以下の項目を編集できます。

```
┌──────────────────────────────────────────────┐
│  氏名       [ 増田 督史          ]              │
│  新しいスタッフID  [ (変更する場合のみ入力) ]      │
│  新しいパスワード  [ (変更する場合のみ入力) ]      │
│  権限       [ 一般スタッフ (staff) ▼ ]          │
│                                               │
│  [ 変更を保存 ]                                 │
│                                               │
│  [ アカウントを凍結 ]  [ アカウントの削除 ]        │
└──────────────────────────────────────────────┘
```

| 項目 | 説明 | 備考 |
|------|------|------|
| 氏名 | スタッフの表示名 | いつでも変更できます |
| 新しいスタッフID | ログイン時に使用するID | 変更する場合のみ入力してください。空欄の場合は変更されません |
| 新しいパスワード | ログイン時に使用するパスワード | 変更する場合のみ入力してください。6文字以上が必要です |
| 権限 | `一般スタッフ (staff)` または `管理者 (admin)` | 権限を変更するとアクセスできる機能が変わります |

3. 変更が完了したら、**変更を保存** ボタンを押してください。

> **注意**  
> - スタッフIDは **半角英数字** で設定してください。  
> - パスワードは **6文字以上** で設定してください。  
> - 自分自身のアカウントの権限を変更する場合は十分にご注意ください。

---

### 3-3. 新しいスタッフアカウントを作成する

スタッフ一覧の下部にある「新規スタッフの登録」フォームから、新しいアカウントを発行できます。

1. スタッフ一覧を下にスクロールし、「新規スタッフの登録」セクションを探します。

```
┌──────────────────────────────────────────────┐
│  新規スタッフの登録                              │
├──────────────────────────────────────────────┤
│  スタッフID (半角英数字) [ 例: amasuda      ]    │
│  氏名                   [ 例: 増田 督史     ]    │
│  初期パスワード (6文字以上) [ ••••••••       ]    │
│  権限                   [ 一般スタッフ (staff) ▼] │
│                                               │
│  [ アカウントを発行する ]                         │
└──────────────────────────────────────────────┘
```

2. 各項目を入力します。

| 項目 | 必須 | 説明 |
|------|:---:|------|
| スタッフID | ○ | ログインに使用するIDです。半角英数字で設定してください |
| 氏名 | ○ | スタッフの表示名です |
| 初期パスワード | ○ | 6文字以上で設定してください |
| 権限 | ○ | `一般スタッフ (staff)` または `管理者 (admin)` を選択します |

3. **アカウントを発行する** ボタンを押してください。
4. アカウントが作成され、スタッフ一覧に追加されます。

> **ヒント**  
> アカウントを発行したら、IDと初期パスワードをスタッフ本人にお伝えください。セキュリティのため、パスワードはメモなどに残さず口頭でお伝えすることを推奨します。

---

### 3-4. アカウントを凍結・凍結解除する

退職や長期休暇などの理由で、一時的にログインを停止させたい場合に、アカウントを凍結できます。

#### 凍結する

1. 該当するスタッフのカードをクリックして展開します。
2. カード下部の **アカウントを凍結** ボタンを押してください。
3. アカウントが凍結され、そのスタッフはシステムにログインできなくなります。

#### 凍結を解除する

1. 凍結済みのスタッフのカード（赤い背景で「凍結済み」と表示されています）をクリックして展開します。
2. **凍結を解除** ボタンを押してください。
3. アカウントが有効に戻り、ログインが再び可能になります。

> **注意**  
> - 自分自身のアカウントを凍結することはできません。  
> - 他の管理者（admin）アカウントを凍結することはできません。  
> - 凍結はアカウントの削除ではありません。データは保持されます。

---

### 3-5. アカウントを削除する

不要になったスタッフアカウントを完全に削除できます。

1. 該当するスタッフのカードをクリックして展開します。
2. **アカウントの削除** ボタンを押してください。
3. 確認のメッセージが表示されます。よろしければ確定してください。

> **注意**  
> - **この操作は取り消せません。** 削除前に、本当に不要であることをよくご確認ください。  
> - 自分自身のアカウントを削除することはできません。  
> - アカウントを一時的に停止したいだけの場合は、削除ではなく「凍結」をご利用ください。

## 4. 月次実績記録表（Excel）の出力

管理者パネルの **スタッフ管理** タブの下部に、Excel出力機能があります。浦安市提出用の所定フォーマットで、月次実績記録表を自動生成・ダウンロードできます。

### 出力手順

1. 管理者パネル > **スタッフ管理** タブを開きます。
2. ページ下部の「月次実績記録表・スケジュール表の出力」セクションを探します。

```
┌──────────────────────────────────────────────┐
│  月次実績記録表・スケジュール表の出力              │
├──────────────────────────────────────────────┤
│  浦安市提出の所定フォーマットを自動生成し、         │
│  Excelファイルとしてダウンロードします。            │
│                                               │
│  対象月  [ 2026-04 ]   [ 出力 (.xlsx) ]         │
└──────────────────────────────────────────────┘
```

3. **対象月** を選択してください（初期値は当月です）。
4. **出力 (.xlsx)** ボタンを押してください。
5. Excel ファイルがダウンロードされます。

### 出力される内容

| 項目 | 説明 |
|------|------|
| ファイル名 | `nicoberi_export_YYYY-MM.xlsx` |
| シート構成 | 児童ごとに1シートが作成されます |
| 記載内容 | 受給者証番号、児童名、来所日、曜日、到着時刻、退出時刻、算定時間、送迎情報 |
| 算定時間の計算 | 実際の利用時間から30分を差し引き、1時間単位に丸めます（30分以上は切り上げ） |

> **ヒント**  
> - 出力前に、当月の打刻データが正しく入力・修正されているかをご確認ください。  
> - テンプレートファイルは `docs/template.xlsx` に格納されています。フォーマットの変更が必要な場合は、このテンプレートを編集してください。

## 5. 過去の打刻履歴を修正する

管理者は、ダッシュボードの **月次履歴** タブから、過去の日付の打刻時刻や送迎フラグを直接編集できます。

> **補足**  
> 一般スタッフは月次履歴の閲覧のみですが、管理者には各レコードに編集フォームが表示されます。

### 打刻時刻を修正する

1. ダッシュボード > **月次履歴** タブを開きます。
2. 修正したい日付のアコーディオンをクリックして開きます。
3. 該当する児童の行に、到着・退出それぞれの時刻入力欄が表示されます。

```
  到着 [ 10:00 ] [更新]    退出 [ 15:30 ] [更新]    [迎え] [送り]
```

4. 正しい時刻を入力し、**更新** ボタンを押してください。

### 送迎フラグを修正する

1. 同じ画面で、**迎え** ボタンまたは **送り** ボタンを押して切り替えてください。
2. ボタンの色がオン（色付き）/オフ（グレー）で状態を示します。

## 6. 予定の削除（管理者限定）

管理者は、ダッシュボードの **予定** タブ（今日・明日のサブタブ）で、各児童カードの「手動で修正・入力」パネル内に **この予定を削除** ボタンが表示されます。

1. 該当する児童のカードで **「手動で修正・入力」** を開きます。
2. パネルの最下部にある赤い **この予定を削除** ボタンを押してください。
3. 予定が削除されます。

> **注意**  
> 削除された予定は元に戻せません。キャンセルとは異なり、記録自体が消去されます。

## 7. 児童の登録・編集・削除

ダッシュボードの **児童・住所** タブで、児童情報の管理を行います。

### 7-1. 新しい児童を登録する

1. **児童・住所** タブを開きます。
2. **新規児童の登録** をクリックして、登録フォームを開きます。
3. 以下の項目を入力してください。

| 項目 | 必須 | 説明 |
|------|:---:|------|
| 姓 | ○ | 児童の姓（漢字） |
| 名 | ○ | 児童の名（漢字） |
| セイ | - | 姓のフリガナ（カタカナ）。一覧の並び順に使われます |
| メイ | - | 名のフリガナ（カタカナ） |
| 生年月日 | - | 生年月日 |
| 性別 | - | 「男」または「女」 |
| 受給者証番号 | - | 受給者証の番号（Excel出力時に使用されます） |
| 支援区分 | - | 支援区分のレベル（数字） |
| スタッフ共有事項 | - | スタッフ全員が閲覧できるメモ欄 |
| 非公開配慮事項 | - | 医療・配慮に関する情報（管理者向け） |

4. 入力が完了したら、**登録を完了する** ボタンを押してください。

> **ヒント**  
> - 「セイ」（フリガナ）を入力しておくと、一覧が五十音順で正しく並びます。  
> - 「非公開配慮事項」は一般スタッフの画面にも表示されますが、特に注意を要する医療・配慮情報を記入する欄としてお使いください。

---

### 7-2. 児童情報を編集する

1. **児童・住所** タブの「登録済み児童名簿」から、編集したい児童をクリックして展開します。
2. 各項目を編集してください。
3. **変更を保存** ボタンを押してください。

---

### 7-3. 児童を削除する

1. 該当する児童のカードをクリックして展開します。
2. 最下部の赤い **この児童を削除** ボタンを押してください。
3. 確認メッセージが表示されます。よろしければ確定してください。

> **注意**  
> - **この操作は取り消せません。**  
> - 児童を削除すると、その児童に関連するスケジュール情報もすべて削除されます。  
> - 削除する前に、必要なデータ（月次実績記録表など）を事前にエクスポートしておくことを強く推奨します。

## 8. 施設（預かり所・学校）の管理

ダッシュボードの **児童・住所** タブで、施設情報の管理を行います。

### 8-1. 施設を登録する

1. **児童・住所** タブを開きます。
2. **預かり所・学校の登録** をクリックして展開します。
3. 以下の項目を入力してください。

| 項目 | 必須 | 説明 |
|------|:---:|------|
| 施設・学校名 | ○ | 施設や学校の名前（例：第一小学校） |
| 住所 | - | 施設の住所 |

4. **追加する** ボタンを押してください。

---

### 8-2. 施設情報を編集する

1. 「登録されている施設」の一覧から、編集したい施設をクリックして展開します。
2. 名前や住所を編集してください。
3. **変更を保存** ボタンを押してください。

---

### 8-3. 施設を削除する

1. 該当する施設のカードをクリックして展開します。
2. **この施設を削除** ボタンを押してください。
3. 確認メッセージが表示されます。よろしければ確定してください。

> **注意**  
> 施設を削除しても、その施設に紐づいている児童のデータは削除されません。児童からは施設情報のみが解除されます。

## 10. 運用上の注意事項

### セキュリティについて

- 管理者パスワードは定期的に変更してください。
- スタッフが退職した場合は、速やかにアカウントを凍結または削除してください。
- パスワードは6文字以上の十分に複雑なものを設定してください。

### データの整合性について

- Excel出力を行う前に、当月の打刻データに漏れや誤りがないかを確認してください。
- 月末には全スタッフに打刻漏れがないか呼びかけを行うことを推奨します。

### 管理者権限の付与について

- 管理者権限を付与するスタッフは必要最小限にしてください。
- 管理者が1名のみの状態は避け、複数名に管理者権限を付与しておくことを推奨します。

## 11. よくあるご質問（FAQ）

### Q. スタッフがパスワードを忘れたと言っています

管理者パネルの該当スタッフの編集画面から、新しいパスワードを設定してください。設定後、新しいパスワードをスタッフに口頭でお伝えください。

### Q. 退職したスタッフのアカウントはどうすればよいですか？

退職したスタッフのアカウントは **凍結** を推奨します。過去のデータを参照する可能性がある場合は、削除ではなく凍結で対応してください。完全に不要な場合は削除も可能です。

### Q. 管理者パネルが表示されません

アカウントの権限が `admin` に設定されているかをご確認ください。別の管理者に権限を確認・変更していただく必要がある場合があります。

### Q. Excel出力でエラーが発生します

以下の点をご確認ください。

- 対象月の打刻データが存在するか
- テンプレートファイル（`docs/template.xlsx`）がリポジトリに正しく配置されているか
- 問題が解消しない場合は、システム開発者にお問い合わせください

### Q. 間違えて児童を削除してしまいました

削除操作は取り消せません。同じ情報で新規に登録し直す必要があります。関連するスケジュールも再登録が必要です。重要なデータは事前にExcelで出力しておくことを推奨します。

### Q. スタッフIDを変更したいのですが

管理者パネルの該当スタッフの編集画面で「新しいスタッフID」欄に新しいIDを入力し、保存してください。変更後は新しいIDでログインする必要があることをスタッフにお伝えください。

## 12. 困ったときは

上記の内容で解決しない問題が発生した場合は、システム開発者にお問い合わせください。

お問い合わせ時には、以下の情報をお伝えいただくとスムーズです。

- どの画面で問題が起きたか（ダッシュボード or 管理者パネル）
- どのような操作をしたか
- 画面に表示されているエラーメッセージ（あればスクリーンショットも）
- 操作した日時
- 影響を受けた児童名やスタッフ名
